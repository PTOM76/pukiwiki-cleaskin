# Cleaskin - PukiWikiスキン
## 導入
README.md, .gitignore以外のすべてのファイルをPukiWikiルートフォルダ直下にコピーする<br />
pukiwiki.ini.phpの
```php
define('SKIN_DIR', 'skin/');
```
を以下のように変更する。
```php
define('SKIN_DIR', 'skin/cleaskin/');
```
